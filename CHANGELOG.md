# Changelog

All notable changes to **vibebar::** (VibeBar) are documented here. Versions follow [Semantic Versioning](VERSIONING.md).

## [1.1.0] — 2026-06-26

### Added
- **Cursor Agent (MCP)** — localhost MCP server on `127.0.0.1:17342`, toolbar panel, connection status, and `mcp.json` snippet copy.
- **Agent Companion** — in-toolbar chat drawer for Cursor agent with model select and chat history.
- **Resource monitor widgets** — floating CPU/memory widgets synced with toolbar layout.
- **Toolbar drag/snap** — improved edge docking and layout handshake across monitors.
- **Docs site** — VitePress documentation on GitHub Pages with guides, features, workflows, and troubleshooting.

### Changed
- README and docs updated for MCP workflow, Session Hub pins, and Ready Check.
- Repo semver aligned to **1.1.0** across all workspace packages and installer artifacts.

### Fixed
- Gitignore no longer excludes notes main-process source files.
- Download link in README (broken nested markdown).

## [1.0.0] — initial release

- Floating Windows overlay toolbar with Prompt Library, Security Audit, Context Packer, Session Hub, Smart Terminal, Code Sync, and related tools.
