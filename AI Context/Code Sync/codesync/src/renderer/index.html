Code Sync

One-way mirror tuned for dev work: only files that actually changed are copied (size + modification time, like a cheap diff). Defaults skip node_modules, .git, Python caches, Rust target, build outputs, and more — add your own globs below. Multiple instances run side by side when you multitask.

Warning: files and folders in each sync folder that are not in its source (and not ignored) will be deleted during sync.
